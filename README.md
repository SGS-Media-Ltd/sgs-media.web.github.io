# 🌍 SGS Media Official Website

Welcome to the official repository for the **SGS Media** website. This is a single-page, high-performance web presence designed to showcase our Christian media services, introduce our team, and facilitate digital commerce.

## 🚀 Live Site
The website is hosted via GitHub Pages and is accessible at:
**[https://sgs-media-hq.github.io](https://sgs-media-hq.github.io)**

---

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
* **Performance:** CSS Scroll-Snap (Carousel), Flexbox/Grid layouts, WebP/Optimised PNG image assets.
* **Hosting:** GitHub Pages (Organisation Tier)
* **Commerce:** Paystack / Flutterwave Integration (Payment Links)
